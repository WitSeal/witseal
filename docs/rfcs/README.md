## Status legend

- **reserved** — number reserved for a future RFC; stub file present
- **draft** — RFC PR open; author iterating before opening for comment
- **open for comment** — minimum 2-week public comment period
- **final comment period** — 1-week FCP with proposed disposition
- **accepted** — merged; may be implemented
- **rejected** — closed; not adopted
- **withdrawn** — closed by author; may be re-opened
